# FarmBeast---ROS2-CrashCourse

Gradivo za začetno delavnico iz ROS 2 v Pythonu.

Delavnica je namenjena popolnim začetnikom in pokriva osnovni razvojni workflow v ROS 2:
- workspace
- package
- node
- timer node
- publisher / subscriber
- topici in message tipi
- service
- osnovni ROS 2 CLI ukazi
- turtlesim demo
- osnovni debugging z `ros2` ukazi in `rqt_graph`


## Komu je gradivo namenjeno

Gradivo je primerno za:
- študente, ki se prvič srečajo z ROS 2
- začetnike, ki želijo razumeti osnovni ROS 2 workflow
- vse, ki želijo narediti prvi Python package in osnovne node-e

## Predpogoji

Pred delavnico je priporočljivo, da imajo udeleženci:
- Ubuntu ali Linux VM
- nameščen ROS 2 Jazzy
- osnovno znanje terminala
- osnovno razumevanje Python-a in OOP

## Cilj delavnice

Cilj delavnice ni iti v globino, ampak zgraditi jasen mentalni model:
- kaj je node
- kaj je topic
- kaj je service
- kako poteka tipičen ROS 2 razvojni proces
